Cube-Level-Midnight

Cube-Level-Midnight (which owes its name to Michael Scott's action movie "Threat Level Midnight" from NBC's "The Office") is a Node web application I built using MongoDB that allows users to create and modify collections of "Magic: The Gathering" cards known as "cubes". The application requires user authentication, sends HTTPS requests to a third-party API (scryfall.com) to get card data, features a blog containing a few lame articles on cube design that I wrote where users can publicly roast me for my terrible ideas, allows users to send and respond to friend requests and draft cards with their friends in real time. I am currently working on building a front end for this application using React.
